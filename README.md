# Bond Yield Calculator

A full-stack bond yield calculator built with React 19, NestJS 10, and shared TypeScript types in a pnpm monorepo.

## Setup

```bash
pnpm install
pnpm --filter @bond/shared build
```

## Development

```bash
pnpm dev          # Start both server and client
pnpm dev:server   # NestJS on http://localhost:3001
pnpm dev:client   # Vite on http://localhost:5173
```

## Testing

```bash
pnpm test
```

## Build

```bash
pnpm build
```
